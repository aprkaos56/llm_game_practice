# 🧅 행복양파, 썩은양파 게임

사용자의 10 마디를 바탕으로 양파의 감정 상태와 행복도를 분석하는 LLM 게임입니다.  
사용자가 입력한 문장을 누적해서 LLM이 양파의 현재 기분을 해석하고, 마지막에는 행복양파인지 썩은양파인지 최종 판정합니다.

---

## 1. 프로젝트 소개

**행복양파, 썩은양파 게임**은 사용자의 감정이 담긴 문장을 입력받아, 이를 바탕으로 양파의 상태를 분석하는 텍스트 기반 감정 게임입니다.

게임은 총 10턴으로 진행되며, 사용자가 한 턴마다 한마디씩 입력하면 양파는 그동안 누적된 말을 종합해 현재 상태를 보여줍니다.  
마지막 10번째 턴이 끝나면 감정 점수표와 함께 최종적으로 양파의 유형을 판정합니다.

이 프로젝트는 LLM의 자연어 이해 능력과 구조화된 출력 기능을 활용해, 단순한 입력-출력 방식이 아닌 **상태 누적형 감정 게임**으로 구현했습니다.

---

## 2. 프로젝트 목표

- 사용자의 입력을 기반으로 감정 흐름을 분석하는 간단한 인터랙티브 게임 구현
- LangChain과 OpenAI 모델을 활용한 구조화된 감정 분석 실습
- 누적 대화를 바탕으로 상태를 변화시키는 텍스트 게임 로직 구현
- 감정 분석 결과를 점수와 서술형 메시지로 함께 제공하는 사용자 친화적 출력 구성
